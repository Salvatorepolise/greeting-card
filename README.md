# Greeting Card – HTML & CSS Project

A simple greeting card project built using only HTML and CSS as part of my learning path on FreeCodeCamp.

## 📌 Overview

This project is a small but important exercise to strengthen my fundamentals in frontend development.  
It focuses on structuring a webpage and styling it with CSS to create a visually clean and centered greeting card.

Even though it is a beginner project, it helped me practice writing clean HTML and improving my understanding of CSS layout and styling.

---

## 🧠 What I Learned

- HTML structure and semantic elements
- CSS styling fundamentals
- Centering elements using Flexbox
- Spacing, margins, and padding
- Visual hierarchy and simple UI design
- Writing clean and readable code

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- Flexbox  
- Git & GitHub Pages (for deployment)
